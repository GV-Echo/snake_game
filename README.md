# Игра "Змейка 2025"

## Установка

1. Убедитесь, что у вас установлен Python 3.x
2. Установите необходимые зависимости:
```sh
pip install -r requirements.txt
```

## Запуск игры

Запустите главный скрипт:
```sh
python main.py
```
Или воспользуйтесь файлами:
 - executable_linux
 - executable_win.exe
